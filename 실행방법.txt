========================================
   용인중앙교회 쁄라 학생회 앱 실행 방법
========================================

[ 처음 설치하는 경우 ]
------------------------------------------
1. Node.js 설치
   - 사이트: https://nodejs.org
   - LTS 버전 다운로드 후 설치

2. 패키지 설치 (최초 1회만 필요)
   - 이 폴더에서 터미널(명령 프롬프트) 열기
   - 아래 명령어 순서대로 입력:

     npm install

     cd server
     npm install

------------------------------------------
[ 매번 실행할 때 ]
------------------------------------------
   "앱시작.bat" 파일을 더블클릭

   → 백엔드 서버 자동 시작 (포트 5000)
   → React 앱 자동 시작 (포트 3000)
   → 브라우저에서 자동으로 앱이 열림
   ※ 처음 실행 시 15~30초 정도 소요됩니다

------------------------------------------
[ 앱 기능 안내 ]
------------------------------------------
   - 주보    : 주일 예배 주보 확인 / 관리자 편집
   - 게시판  : 공지 및 게시글 작성
   - 성경 검색: 구절 찾기 / 키워드 검색

------------------------------------------
[ 관리자 모드 ]
------------------------------------------
   - 우측 상단 "관리자" 버튼 클릭 후 로그인
   - 관리자 ON 상태에서 주보 편집 및 DB 저장 가능

------------------------------------------
[ 문의 ]
------------------------------------------
   문제 발생 시 개발자에게 문의하세요.
